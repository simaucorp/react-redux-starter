A react starter kit with opinionated project structure.

- Add environment and api endpoint on `.env` file (you can duplicate the `.env.example` and edit it according to your need)

- Install dependencies

```bash
$ npm i
```

- Run the project

```bash
$ npm start
```

- Build the project

```bash
$ npm run build
```

- Test the project

```bash
$ npm run test
```
